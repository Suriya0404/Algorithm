Snowflake Data Quality Framework

Framework to proactively monitor and validate the data in Snowflake database to deliver high quality reliable data to the stakeholders. Below objectives are achieved using this framework

Data reliability - Routine checks to verify the scheduled jobs are executed daily

Data accuracy - Duplicate key check and other checks to verify the record count matches between source system and target system are performed to maintain data accuracy.

Detect anomaly - Compare the data volume of current load with prior loads to detect anomalies in data, source data issue and ETL issue.

Metrics monitoring - Identify the trend in metrics by capturing the daily metric values and comparing it with historical values. Also check variance in metrics are within the threshold to detect the anomalies.
